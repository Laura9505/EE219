How to run the code:
    The following instructions are for windows pc. Running on other OS hasn't been tested.

    Run cmd.exe as administrator.
    Set path to current .py files folder.
    Type "python 1.py" to run part a of the project.
    Type "python 2a.py" to run part 2a and 2d question i of the project.
    Type "python 2b.py" to run part 2b of the project. When executing this, it will display lots of warnning, just ingore those warnnings.
    Type "python 2c.py" to run part 2c of the project.
    Type "python 2d.py" to run part 2d question ii of the project.
    Type "python 2e.py" to run part 2e of the project.
    You may need Anaconda to execute code correctly.
    
    Note: You may need to close figure windows to let the program continue running.
          Please keep network_backup_dataset.csv, codes and pics file folder in the same file folder.
